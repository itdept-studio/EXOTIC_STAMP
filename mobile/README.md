# Mobile App

Flutter app cho Metro Stamp mobile experience.

## Initial scope

- Auth
- Station list/detail
- QR scan
- Stamp book
- Rewards

## Key folders

- `lib/app/`: app shell, router, theme, config.
- `lib/core/`: shared services, constants, widgets.
- `lib/features/`: feature-based modules.
- `lib/shared/`: DTO, models va reusable widgets.
